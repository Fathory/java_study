# java_study

## 구구단 구현

- 명령어 및 단축키 사용법
- 입력 및 출력
- 변수선언, 반복문, 조건문 사용
- 배열, 메소드, 클래스를 사용한 프로그래밍

### 2, 3단 구현
* 구구단 2단, 3단을 구현한다.
* 각각의 값을 계산 후 출력한다.
* System.out.println();을 이용하여 출력한다.

### 4, 5단 구현
* 구구단 4단, 5단을 구현한다.
* 변수를 사용한다. 
* 숫자를 입력받아 결과를 출력한다.
** 구구단에서 4단과 5단을 계산한 결과 값을 변수에 저장한 후 저장한 변수 값을 출력한다.
** 4단과 5단은 한번 결정하면 바뀌지 않는 값이다. 따라서 4단과 5단에 해당하는 상수 값으로 만든 후 프로그램을 구현한다.
** 예를 들어 4단은 4 곱하기 1 계산 결과 값을 i라는 변수에 저장하고, i라는 변수에 저장한 결과 값을 출력한다.
** 사용자가 입력한 숫자 값에 해당하는 단을 출력한다.
** 예를 들어 사용자가 4라는 숫자를 입력하면 4단 전체를 계산해 출력한다.


### 6, 7단 구현
* 구구단 6단, 7단을 구현한다.
* 반복문을 사용한다. 
* 숫자를 입력받아 결과를 출력한다.
* 반복문을 활용하여 단순 반복작업한 내용의 코드를 축소하여 수행한다.


### 8, 9단 구현
* 구구단 8단, 9단을 구현한다. 
* 숫자를 입력받아 결과를 출력한다.
* 조건문을 활용하여 2 이상 9 이하의 숫자만 입력받도록 한다.
